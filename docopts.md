# Plugin ops ai

AI tasks to generate code.

## Synopsis

```text
Usage:
  ai new <action> [<package>]
  ai ask [<args>...]
  ai cli [--ide]
  ai chat <action> [<args>...]
```

# Options:

- `--ide`  enable ide mode for the CLI
- `--model=<model>` select the model, use  `list` as model for a list of the available models
